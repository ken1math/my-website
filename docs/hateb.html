Qiita はてなブックマークチェッカー

■ 使い方

  • Qiitaユーザー名を入力し、「チェック開始」をクリック。
  • 最新100記事のはてブ数、いいね数、投稿日を取得。
  • テーブルのヘッダーをクリックで昇順・降順ソート。

■ 注意事項

  • Qiita API と はてなブックマーク API の利用制限にご注意ください。
  • 1回の呼び出しで取得できる記事は最大100件です。
  • モダンブラウザ推奨（IE非対応の場合あり）。
  • はてブ取得はJSONPを使用。設定によりブロックされる場合があります。
